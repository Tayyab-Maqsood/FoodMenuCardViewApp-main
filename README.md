# Food Menu CardView App

A simple Android application developed in Android Studio using Java, RecyclerView, and CardView.

## Features
- Attractive CardView design
- RecyclerView implementation
- Food images with names and prices
- Scrollable food menu
- Simple and user-friendly interface

## Technologies Used
- Java
- Android Studio
- RecyclerView
- CardView
- XML Layouts

## App Screens
The application displays different food items such as:
- Burger
- Pizza
- Fries
- Pasta
- Sandwich

Each card contains:
- Food Image
- Food Name
- Description
- Price

## Purpose
This project was created as an assignment for the Mobile Application Development course.
